# Urban Palate - Demo Restaurant Website

**Urban Palate** is a modern, elegant restaurant web app built with **Django** and **Tailwind CSS**, showcasing a responsive landing page, authentication, and a premium UI/UX design.  

---

## Features
- Dual-panel landing page with responsive design.
- User login & registration; staff/admin redirects.
- Custom 404 page with split-screen layout.
- CTA buttons, info panels, and subtle animations.
- Demo-only: no real bookings or transactions.

---

## Tech Stack
- **Backend**: Python 3.x, Django 4.x
- **Frontend**: HTML, Tailwind CSS
- **Database**: SQLite (default)
- **Fonts**: Playfair Display + Inter

---

## Setup
1. Clone the repo:
```bash
git clone https://github.com/ORLAND0HENRY/urban_restaurant.git
cd urban_restaurant

2.Install dependencies:

pip install -r requirements.txt


3.Run migrations and start the server:

python manage.py migrate
python manage.py runserver


4.Visit: http://127.0.0.1:8000
